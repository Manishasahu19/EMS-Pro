# 🏢 EMS Pro - Employee Management System

EMS Pro is a secure and modern web application built using **Django (Python)** and **Bootstrap 5** to manage employee records efficiently with full CRUD functionality.

## Features
*  **Secure Auth:** Protected backend using Django's built-in login/logout system.
* **Modern UI:** Premium Dark Mode dashboard with interactive Bootstrap 5 cards.
* **Advanced Search:** Instant filtering by Name, Department, or Role using Django `Q` objects.
*  **Relational DB:** Structured SQLite database connecting Employee, Department, and Role tables.

##  Tech Stack
* **Backend:** Python, Django Framework
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Database:** SQLite (Django ORM)
* **Static Files:** WhiteNoise
